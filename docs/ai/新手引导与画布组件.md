### 二、⭐ Driver.js — 推荐首选

> 最轻量、最灵活、TypeScript 原生、框架无关

- **网址**：https://driverjs.com
- **GitHub**：https://github.com/kamranahmedse/driver.js
- **npm**：`driver.js`

#### 核心能力

| 功能 | 说明 |
|------|------|
| **产品引导** | 多步骤 Tour，高亮目标元素 + 弹出说明 |
| **高亮聚焦** | 单独高亮任意元素（非 Tour 场景） |
| **SVG 遮罩** | 平滑动画的镂空遮罩，可自定义颜色/透明度 |
| **弹出框定位** | 自动根据空间调整位置（上/下/左/右） |
| **进度显示** | 内置步骤进度 (Step 2 of 5) |
| **异步步骤** | 支持异步操作（如等待数据加载后才进入下一步） |
| **退出确认** | 用户关闭时可弹出确认 |
| **Hooks** | 每步的 onHighlightStarted / onHighlighted / onDeselected |
| **键盘导航** | ← → ESC 导航 |

#### React Hook 封装

```tsx
// hooks/useProductTour.ts
import { driver, type DriveStep } from "driver.js";
import "driver.js/dist/driver.css";
import { useCallback, useRef } from "react";

export function useProductTour(steps: DriveStep[]) {
  const driverRef = useRef<ReturnType<typeof driver>>();

  const start = useCallback(() => {
    driverRef.current = driver({
      showProgress: true,
      animate: true,
      steps,
      onDestroyed: () => {
        localStorage.setItem("tour-completed", "true");
      },
    });
    driverRef.current.drive();
  }, [steps]);

  const stop = useCallback(() => {
    driverRef.current?.destroy();
  }, []);

  return { start, stop };
}

// 使用
function Workbench() {
  const { start } = useProductTour(workbenchSteps);

  useEffect(() => {
    const completed = localStorage.getItem("tour-completed");
    if (!completed) start();
  }, [start]);

  return <div>...</div>;
}
```

### 三、React Joyride — React 原生方案

> 最流行的 React 专用引导库

- **网址**：https://react-joyride.com
- **npm**：`react-joyride`

```tsx
import Joyride, { type Step } from "react-joyride";

const steps: Step[] = [
  {
    target: "#sidebar",
    content: "在这里浏览项目",
    placement: "right",
  },
  {
    target: ".compile-btn",
    content: "点击编译你的设计",
    placement: "bottom",
  },
];

function App() {
  const [run, setRun] = useState(true);

  return (
    <>
      <Joyride
        steps={steps}
        run={run}
        continuous
        showProgress
        showSkipButton
        styles={{
          options: {
            primaryColor: "#4f7cff",
            zIndex: 10000,
          },
        }}
        callback={(data) => {
          if (data.status === "finished" || data.status === "skipped") {
            setRun(false);
          }
        }}
      />
      <MainApp />
    </>
  );
}
```

**优势**：React 组件 API、受控模式、回调丰富  
**劣势**：体积较大、样式自定义不如 Driver.js 灵活

### 四、OnboardJS — 无头架构 (新锐)

> 2026 年新兴方案，无头 (Headless) 状态机 + React Hooks

- **网址**：https://onboardjs.com
- **npm**：`@onboardjs/core` + `@onboardjs/react`

```tsx
import { useOnboard } from "@onboardjs/react";

const steps = [
  { id: "welcome", data: { title: "欢迎", description: "..." } },
  { id: "feature", data: { title: "核心功能", target: "#editor" } },
];

function Tour() {
  const { currentStep, next, prev, isActive } = useOnboard({ steps });

  if (!isActive || !currentStep) return null;

  // 完全自定义 UI — 你决定怎么渲染
  return (
    <div className="fixed bottom-4 right-4 rounded-xl bg-gray-900 p-4 shadow-xl">
      <h3>{currentStep.data.title}</h3>
      <p>{currentStep.data.description}</p>
      <div className="flex gap-2">
        <button onClick={prev}>上一步</button>
        <button onClick={next}>下一步</button>
      </div>
    </div>
  );
}
```

**优势**：完全自定义 UI、内置分析埋点、状态机架构  
**劣势**：新项目，生态尚小，不提供高亮遮罩（需自行实现）

### 五、新手引导选型

```
需要快速集成 + 高亮遮罩 + 最小体积？
  → Driver.js ⭐ (本项目首推)

需要 React 组件 API + 受控状态？
  → React Joyride

需要完全自定义 UI + 分析埋点？
  → OnboardJS (无头架构)

需要跨框架复用？
  → Shepherd.js (Vue/React/Angular/Ember)

只需要简单引导 + 不介意付费？
  → Intro.js
```

### 六、画布库分类与对比

#### 按用途分类

| 类别 | 代表库 | 适用场景 |
|------|--------|----------|
| **节点流程图** | React Flow (xyflow) | 工作流编辑器、数据流图、EDA 原理图 |
| **无限白板** | tldraw, Excalidraw | 手绘白板、协作画板、草图 |
| **2D 图形编辑** | Fabric.js, Konva.js | 图片编辑器、设计工具、图形标注 |
| **高性能渲染** | PixiJS, react-pixi | 游戏、数据可视化、大量图形 |
| **图表/关系图** | D3.js, Mermaid, Cytoscape | 数据可视化、网络拓扑 |

#### 综合对比

| 库 | 渲染方式 | React 集成 | 拖拽 | 缩放/平移 | 自定义节点 | 协作 | 体积 | 定价 |
|----|---------|-----------|------|----------|-----------|------|------|------|
| **React Flow** | SVG + HTML | ✅ 原生 | ✅ | ✅ | ✅ React 组件 | ❌ (自行实现) | ~40KB | 开源 / Pro 订阅 |
| **tldraw** | Canvas | ✅ SDK | ✅ | ✅ 无限画布 | ✅ 自定义形状 | ✅ 内置 | ~200KB | 商业许可 |
| **Excalidraw** | Canvas | ✅ React 组件 | ✅ | ✅ 无限画布 | ⚠️ 有限 | ✅ 内置 | ~180KB | MIT 开源 |
| **Konva.js** | Canvas 2D | ✅ react-konva | ✅ | ✅ | ✅ 声明式形状 | ❌ | ~30KB | MIT 开源 |
| **Fabric.js** | Canvas 2D | ⚠️ 手动集成 | ✅ | ✅ | ✅ 对象模型 | ❌ | ~90KB | MIT 开源 |
| **PixiJS** | WebGL/WebGPU | ⚠️ react-pixi | ✅ | ✅ | ✅ | ❌ | ~50KB | MIT 开源 |

### 七、⭐ React Flow (xyflow) — 节点/流程图首推

> **EDA 工具的最佳画布选择**：节点式 UI、连线、自定义节点

- **网址**：https://reactflow.dev
- **团队**：https://xyflow.com
- **npm**：`@xyflow/react`
- **GitHub Stars**：~26K

#### 核心能力

| 功能 | 说明 |
|------|------|
| **节点 (Nodes)** | 完全自定义 React 组件作为节点 |
| **边 (Edges)** | 直线/贝塞尔/阶梯/自定义路径，可交互 |
| **连接 (Handles)** | 从节点端口拖拽创建连线 |
| **拖拽排列** | 节点自由拖拽定位 |
| **缩放/平移** | 无限画布 + Minimap 小地图 |
| **选择** | 单选/多选/框选 |
| **撤销/重做** | 结合 Zustand/XState 实现 |
| **布局算法** | 集成 dagre / elkjs 自动布局 |
| **导出** | 截图 / 导出为图片 |
| **控件** | Controls (缩放控件) / Background (网格/点阵) / Minimap |

#### 安装

```bash
npm install @xyflow/react
```

#### 基本使用

```tsx
import {
  ReactFlow,
  MiniMap,
  Controls,
  Background,
  useNodesState,
  useEdgesState,
  addEdge,
  type Connection,
} from "@xyflow/react";
import "@xyflow/react/dist/style.css";

const initialNodes = [
  { id: "1", position: { x: 0, y: 0 }, data: { label: "Input" } },
  { id: "2", position: { x: 200, y: 100 }, data: { label: "Process" } },
  { id: "3", position: { x: 400, y: 0 }, data: { label: "Output" } },
];

const initialEdges = [
  { id: "e1-2", source: "1", target: "2" },
  { id: "e2-3", source: "2", target: "3" },
];

function FlowEditor() {
  const [nodes, setNodes, onNodesChange] = useNodesState(initialNodes);
  const [edges, setEdges, onEdgesChange] = useEdgesState(initialEdges);

  const onConnect = useCallback(
    (params: Connection) => setEdges((eds) => addEdge(params, eds)),
    [setEdges],
  );

  return (
    <div style={{ width: "100%", height: "100vh" }}>
      <ReactFlow
        nodes={nodes}
        edges={edges}
        onNodesChange={onNodesChange}
        onEdgesChange={onEdgesChange}
        onConnect={onConnect}
        fitView
      >
        <Controls />
        <MiniMap />
        <Background variant="dots" gap={12} size={1} />
      </ReactFlow>
    </div>
  );
}
```

#### 自定义节点（EDA 场景）

```tsx
import { Handle, Position, type NodeProps } from "@xyflow/react";

function GateNode({ data }: NodeProps) {
  return (
    <div className="rounded-lg border border-blue-500/30 bg-gray-900/90 px-4 py-2 shadow-lg">
      <Handle type="target" position={Position.Left} />
      <div className="text-xs text-blue-400">{data.type}</div>
      <div className="text-sm font-medium text-white">{data.label}</div>
      <Handle type="source" position={Position.Right} />
    </div>
  );
}

// 注册
const nodeTypes = { gate: GateNode };
<ReactFlow nodeTypes={nodeTypes} ... />
```

### 八、tldraw — 无限白板 SDK

> 用于嵌入应用内的白板/画布体验

- **网址**：https://tldraw.dev
- **GitHub**：https://github.com/tldraw/tldraw
- **npm**：`tldraw`

#### 核心能力

- 无限画布 + 手绘风格
- 内置工具：画笔/箭头/形状/文字/图片/便签
- 自定义形状 (Custom Shapes) + 自定义工具 (Custom Tools)
- 实时协作（多人同步）
- 高性能渲染（Canvas）
- 导出为图片/SVG
- 商业用途需要许可证密钥

#### 嵌入使用

```tsx
import { Tldraw } from "tldraw";
import "tldraw/tldraw.css";

function Whiteboard() {
  return (
    <div style={{ width: "100%", height: "100vh" }}>
      <Tldraw />
    </div>
  );
}
```

**适合**：需要嵌入白板/自由画图的场景，如头脑风暴、架构草图

### 十、Konva.js / react-konva — 2D 图形绘制

> 声明式 Canvas 2D 框架，适合图形编辑器

- **网址**：https://konvajs.org
- **React 绑定**：`react-konva`

#### 核心能力

- 图层系统 (Stage → Layer → Shape)
- 内置形状：Rect / Circle / Line / Text / Image / Arrow / Star / Ring 等
- 拖拽 / 缩放 / 旋转 / 变形
- 事件系统（click / hover / drag 等）
- 滤镜 / 缓存 / 裁切
- 动画 (Tween)
- 导出为图片/数据 URL

#### 使用示例

```tsx
import { Stage, Layer, Rect, Circle, Text } from "react-konva";

function CanvasEditor() {
  return (
    <Stage width={800} height={600}>
      <Layer>
        <Rect
          x={20} y={20} width={100} height={50}
          fill="#4f7cff" cornerRadius={8}
          draggable
        />
        <Circle
          x={200} y={100} radius={30}
          fill="#2bc5ff"
          draggable
        />
        <Text
          x={300} y={50} text="Hello EDA"
          fontSize={18} fill="#e8eef7"
        />
      </Layer>
    </Stage>
  );
}
```

**适合**：自定义图形编辑器、标注工具、简单的电路图绘制（<20K 元素）  
**注意**：20K+ 元素性能下降，需要虚拟化或改用 PixiJS (WebGL)

### 十一、Fabric.js — 对象模型画布

> 最成熟的 Canvas 对象模型库

- **网址**：https://fabricjs.com
- **特点**：丰富的对象模型（每个图形都是可操作的对象）

#### vs Konva.js

| 维度 | Fabric.js | Konva.js |
|------|----------|---------|
| React 集成 | ⚠️ 需手动 ref | ✅ react-konva 声明式 |
| 对象模型 | 更强（SVG 导入/导出、序列化） | 图层式 |
| 图片处理 | 内置滤镜、裁切、蒙版 | 基础滤镜 |
| 体积 | ~90KB（较重） | ~30KB |
| 适合 | 设计编辑器、图片编辑 | 交互式 UI、标注 |
| 周下载 | ~500K | ~400K |

**2026 总结**：Fabric.js 适合设计编辑器，Konva.js 适合交互式 UI（React 项目优先选 Konva）

---

### 十二、PixiJS — 高性能 WebGL 渲染

> 当 Canvas 2D 性能不够时的选择

- **网址**：https://pixijs.com
- **React 绑定**：`@pixi/react`
- **特点**：WebGL + WebGPU 渲染，万级元素无压力

**适合**：
- 大量图形元素（>10K）
- 需要 60fps 动画
- 数据可视化 / 粒子效果 / 游戏

**不适合**：
- 简单图形编辑（杀鸡用牛刀）
- 需要 DOM 交互（PixiJS 完全脱离 DOM）

### 十三、画布选型决策树

```
做节点/连线/流程图/数据流？
  → React Flow (xyflow) ⭐ (本项目首推)

做自由白板/手绘草图？
  ├─ 需要产品级 SDK + 自定义形状 → tldraw
  └─ 需要完全开源 (MIT) → Excalidraw

做图形编辑器（矩形/圆/文字/图片拖拽编辑）？
  ├─ React 项目 → Konva.js + react-konva
  └─ 需要强对象模型/SVG 导入 → Fabric.js

需要渲染万级图形元素？
  → PixiJS (WebGL)

做数据可视化图表？
  → D3.js / ECharts / Recharts (不是本文范围)
```
