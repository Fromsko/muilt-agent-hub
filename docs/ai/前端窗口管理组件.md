## 二、⭐ Dockview — IDE 级窗口管理 (首推)

> **最完整的方案，覆盖 EDA 工具所需的全部布局能力**

- **网址**：https://dockview.dev
- **GitHub**：https://github.com/mathuo/dockview
- **npm**：`dockview-react`
- **特点**：零依赖、TypeScript 原生

### 核心能力

| 功能 | 说明 |
|------|------|
| **标签页 (Tabs)** | 面板分组，标签可拖拽重排 |
| **拖放 (Drag & Drop)** | 标签拖到任意边缘吸附、合并到已有组 |
| **浮动面板 (Floating)** | 从网格中脱离为自由浮动覆盖层，多个同时存在 |
| **弹出窗口 (Popout)** | 移到独立浏览器窗口，仍与主布局连接 |
| **分割视图 (Split)** | 水平/垂直任意分割 |
| **序列化** | `api.toJSON()` / `api.fromJSON()` 保存/恢复完整布局 |
| **主题** | CSS 变量全定制，内置多套主题 |

### 安装

```bash
npm install dockview-react
```

### 基本使用

```tsx
import {
  DockviewReact,
  DockviewReadyEvent,
  IDockviewPanelProps,
} from "dockview-react";
import "dockview-react/dist/styles/dockview.css";

// 面板组件
function EditorPanel(props: IDockviewPanelProps) {
  return <div className="p-4">编辑器面板: {props.params.title}</div>;
}

function PropertiesPanel(props: IDockviewPanelProps) {
  return <div className="p-4">属性面板</div>;
}

// 注册面板
const components = {
  editor: EditorPanel,
  properties: PropertiesPanel,
};

// 布局初始化
function onReady(event: DockviewReadyEvent) {
  const api = event.api;

  // 添加面板
  api.addPanel({
    id: "editor-1",
    component: "editor",
    params: { title: "main.v" },
  });

  api.addPanel({
    id: "properties",
    component: "properties",
    position: { referencePanel: "editor-1", direction: "right" },
  });
}

function Workbench() {
  return (
    <DockviewReact
      className="h-full"
      components={components}
      onReady={onReady}
    />
  );
}
```

### 布局持久化

```tsx
// 保存
const layout = api.toJSON();
localStorage.setItem("dockview-layout", JSON.stringify(layout));

// 恢复
const saved = JSON.parse(localStorage.getItem("dockview-layout")!);
api.fromJSON(saved);
```

### 浮动面板

```tsx
api.addPanel({
  id: "floating-panel",
  component: "editor",
  floating: {
    width: 400,
    height: 300,
    position: { left: 100, top: 100 },
  },
});
```

### 弹出窗口（独立浏览器窗口）

```tsx
// 将面板移到独立浏览器窗口
api.addPopoutGroup(panel);
```

## 三、WinBox.js / react-winbox — Windows 风格窗口

> **最接近 Windows 桌面的体验**：标题栏 + 最小化/最大化/关闭按钮

- **WinBox.js 官网**：https://nextapps-de.github.io/winbox/
- **react-winbox GitHub**：https://github.com/RickoNoNo3/react-winbox
- **npm**：`react-winbox`

### 核心能力

| 功能 | 说明 |
|------|------|
| 标题栏 | 自定义标题 + 图标 |
| 最小化 | 缩小到任务栏（可自定义位置） |
| 最大化 | 全屏铺满容器 |
| 全屏 | 浏览器全屏 (Fullscreen API) |
| 关闭 | 带回调确认 |
| 拖拽 | 标题栏拖拽移动 |
| 缩放 | 四边 + 四角缩放 |
| 模态 | 阻断背景交互 |
| 主题 | CSS 类切换 |

### 使用示例

```tsx
import WinBox from "react-winbox";
import "winbox/dist/css/winbox.min.css";

function App() {
  const [windows, setWindows] = useState([
    { id: 1, title: "编辑器", show: true },
    { id: 2, title: "属性面板", show: true },
  ]);

  return (
    <>
      {windows
        .filter((w) => w.show)
        .map((w) => (
          <WinBox
            key={w.id}
            title={w.title}
            width={500}
            height={400}
            x="center"
            y="center"
            onClose={() =>
              setWindows((prev) =>
                prev.map((win) =>
                  win.id === w.id ? { ...win, show: false } : win,
                ),
              )
            }
          >
            <div className="p-4">窗口内容: {w.title}</div>
          </WinBox>
        ))}
    </>
  );
}
```

### WinBox.js 原生 API

```js
const win = new WinBox({
  title: "编辑器",
  width: "50%",
  height: "50%",
  x: "center",
  y: "center",
  background: "#0f1728",
  border: "1px solid rgba(255,255,255,0.1)",
  class: ["no-full", "my-theme"],
});

win.minimize();   // 最小化
win.maximize();   // 最大化
win.fullscreen(); // 全屏
win.restore();    // 恢复
win.close();      // 关闭
win.move(100, 100);     // 移动
win.resize(800, 600);   // 缩放
win.setTitle("新标题");  // 改标题
win.setBackground("#1a1a2e"); // 改背景
```

## 五、react-mosaic — Tiling 平铺窗口

> VSCode / Bloomberg Terminal 风格的平铺布局

- **GitHub**：https://github.com/nomcopter/react-mosaic
- **Demo**：https://nomcopter.github.io/react-mosaic/
- **npm**：`react-mosaic-component`

### 特点

- 二叉树结构管理布局
- 拖拽分割/合并
- 全面受控 (controlled)
- TypeScript 原生
- 无最小化/最大化/浮动（纯平铺）

### 使用

```tsx
import { Mosaic, MosaicWindow } from "react-mosaic-component";
import "react-mosaic-component/react-mosaic-component.css";

function App() {
  return (
    <Mosaic<string>
      renderTile={(id, path) => (
        <MosaicWindow path={path} title={`窗口 ${id}`}>
          <div>内容 {id}</div>
        </MosaicWindow>
      )}
      initialValue={{
        direction: "row",
        first: "editor",
        second: {
          direction: "column",
          first: "properties",
          second: "console",
        },
        splitPercentage: 70,
      }}
    />
  );
}
```

## 六、react-rnd — 底层基元

> 只提供**拖拽 + 缩放**，不含窗口 chrome（标题栏/按钮），需自行搭建

- **GitHub**：https://github.com/bokuweb/react-rnd
- **npm**：`react-rnd`

### 使用

```tsx
import { Rnd } from "react-rnd";

<Rnd
  default={{ x: 0, y: 0, width: 320, height: 200 }}
  minWidth={200}
  minHeight={120}
  bounds="parent"
>
  <div className="rounded-lg border border-white/10 bg-gray-900 shadow-xl">
    {/* 自定义标题栏 */}
    <div className="flex items-center justify-between px-3 py-2 border-b border-white/10">
      <span className="text-sm">窗口标题</span>
      <div className="flex gap-1">
        <button>—</button>
        <button>□</button>
        <button>✕</button>
      </div>
    </div>
    {/* 内容 */}
    <div className="p-4">窗口内容</div>
  </div>
</Rnd>
```

### 适合

- 需要完全自定义窗口外观
- 只需要 1-2 个浮动面板
- 不需要复杂的 docking/tiling

## 七、react-grid-layout — 网格仪表盘

> 适合 Dashboard 场景，不是传统窗口

- **GitHub**：https://github.com/react-grid-layout/react-grid-layout
- **npm**：`react-grid-layout`

### 特点

- 可拖拽 + 可缩放的网格卡片
- 响应式断点
- 自动排列 / 碰撞检测
- 序列化 / 持久化布局
- 不支持浮动 / 最小化 / 最大化

### 适合

- 数据仪表盘
- Widget 管理
- Grafana / Metabase 风格界面

## 八、选型决策

```
需要完整的 IDE 级布局（标签页 + 拖拽 + 浮动 + 弹出窗口）？
  → Dockview ⭐ (本项目首推)

需要 Windows 桌面风格（标题栏 + 最小化/最大化/关闭）？
  → WinBox.js / react-winbox

需要 VSCode 风格纯平铺？
  → react-mosaic

只需要拖拽+缩放，自己画窗口 chrome？
  → react-rnd

仪表盘网格卡片？
  → react-grid-layout
```

### 本项目推荐组合

```
┌─────────────────────────────────────────────────┐
│  主布局: Dockview                                │
│  → 工作台的面板管理（编辑器/属性/控制台/日志）   │
│  → 标签页 + 分割 + 拖拽重排                     │
│  → 布局序列化到 localStorage                     │
└─────────────────────────────────────────────────┘
                    +
┌─────────────────────────────────────────────────┐
│  浮动窗口: Dockview Floating 或 react-winbox    │
│  → 弹出式的属性编辑器、AI 聊天窗口              │
│  → 最小化到底栏 / 最大化铺满                     │
└─────────────────────────────────────────────────┘
```

**为什么选 Dockview**：
1. 零依赖，体积小
2. 原生支持 React（也支持 Vue/Angular）
3. 浮动面板 + 弹出窗口 + 序列化一站式搞定
4. 文档质量高，有丰富的 Example
5. CSS 变量主题化，契合暗色 EDA 工具风格
6. 活跃维护（2026 年仍在更新）
